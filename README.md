# Clube da Leitura

## Introdução
Uma ferramenta de controle para otimizar a gestão dos processos do Clube da Leitura.

## Funcionadlidades
Permite realizar: 
- MENU AMIGOS
	- inserção de novos amigos
	- edição de amigos já cadastrados
	- excluir amigos já cadastrado
	- visualizar amigos cadastrados
	- visualizar os empréstimos do amigo
- MENU CAIXAS
	-  cadastrar novas caixas 
	-  editar caixas existentes 
	-  excluir caixas
	-  visualizar todas as caixas
- MENU REVISTAS
	- cadastrar novas revistas 
	- editar revistas existentes
	- excluir revistas
	- visualizar todas as revista
	- mostrar o status atual (disponível/emprestada/reservada)
- MENU EMPRÉSTIMO 
	- registrar novos empréstimos
	- registrar devoluções
	- visualizar empréstimos abertos e fechados

## Requisitos

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.
---
## Como Usar

#### Clone o Repositório
```
https://github.com/AlexAraldi/ClubeDaLeitura.ConsoleApp.git
```

#### Navegue até a pasta raiz da solução
```
cd ClubeDaLeitura
```

#### Restaure as dependências
```
dotnet restore
```

#### Navegue até a pasta do projeto
```
cd ClubeDaLeitura.ConsoleApp
```

#### Execute o projeto
```
dotnet run
```
